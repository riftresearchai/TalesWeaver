# TalesWeaver

**Weaving words into stories.**

TalesWeaver is an experimental small language model focused on story completion and narrative generation. Built using the TinyStories dataset, it explores the capabilities of compact transformer-based models in learning language patterns, maintaining context, and generating coherent stories.

## Overview

The goal of TalesWeaver is to investigate how small language models can learn storytelling abilities from carefully curated datasets. The project focuses on training, evaluating, and experimenting with efficient generative AI architectures.
